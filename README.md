# sperm-analysis

# Filipe Miranda de Oliveira

# Contagem automatizada por meio de visão computacional de espermatozoides, a partir de imagens de microscopia.

Na análise são contados apenas espermatozoides, não sendo contabilizada
célula do tipo epitelial que se descamou e veio junto com o sêmen.

Resultado de saída é de uma imagem contendo: 
número arquivo e número de espermatozoides na imagem
ex: 12 - sperm numbers 10.tif 
imagem com identificador 12 contendo total de 10 espermatozoides
